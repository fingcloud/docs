# استقرار برنامه Node در فینگ

## نصب cli

حتما قبل از شروع استقرار برنامه‌ی خود بر روی سکوی ابری ** فینگ** ، از نصب بودن [فینگ CLI](/cli/installation) مطمئن شوید.

```
$ npm i -g @fingcloud/cli
```

## ورود با cli

با وارد کردن دستور زیر، **ایمیل** و **رمز عبور** حساب کاربری خود در فینگ را وارد نمایید.

```
$ fing login
```

# مراحل استقرار:

## ۱: ایجاد برنامه

از قسمت [داشبورد](https://dashboard.fing.ir) وارد بخش **برنامه‌ها** شوید و بر روی گزینه **ساخت برنامه جدید** کلیک کنید. سپس نام برنامه مورد نظر خود را وارد کرده و پلتفرم خود را **node** قرار دهید و همچنین پلن مورد نیاز خود را انتخاب کرده و بر روی دکمه **ایجاد برنامه** کلیک کنید تا برنامه شما ساخته شود.

## ۲: package.json

موقعی که یک پروژه‌ی nodeJs را شروع می‌کنید با وارد کردن دستور `npm init` یا `yarn init` فایلی با نام `package.json` ساخته می‌شود. برای اینکه **فینگ** بتواند برنامه شما را اجرا کند، باید در قسمت `scripts` فیلدی با نام `start` تعریف کنید. و داخل آن دستوری که برنامه شما به وسیله‌ی آن اجرا می‌شود را قرار دهید

```javascript
{
  "name": "node-app",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "license": "MIT",
  "scripts": {
    "start": "node index.js",
    ...
  },
  "dependencies": {
    ...
  },
}
```

## ۳: دستور استقرار

در مسیر اصلی پروژه‌ی خود، دستور زیر را وارد کنید تا اولین استقرار انجام شود.

```
$ fing up
```

- با اجرای دستور `fing up` فینگ ‌CLI به صورت خودکار تشخیص می‌دهد که برنامه شما **node** می‌باشد و عملیات استقرار را آغاز می‌کند.
- شما می‌توانید موقع اجرای دستور بالا، نام برنامه و پلتفرم آن را نیز وارد کنید.

```
$ fing up --app=node-app --platform=node
```

### فایل fing.yaml

برای اینکه لازم نباشد در هر استقرار نام **برنامه** ، **پلتفرم** و **پورت** آن را مشخص کنید. می‌توانید در ریشه اصلی برنامه‌ی خود فایلی با نام `fing.yaml` بسازید و داخل آن نام برنامه و پلتفرمی که از آن استفاده کردید و پورتی که برنامه‌ی شما بر روی‌ آن `listen` می‌کند (اصطلاحا گوش می‌کند) را قرار دهید.

```yaml
app: "node-app"
platform: "node"
port: 3000
```

### فایل‌هایی که برای استقرار برنامه لازم است

برای اینکه برنامه‌ی node شما بتواند بر روی سکوی ابری فینگ مستقر شود، حتما باید دو فایل `package.json` و `package-lock.json` یا `yarn.lock` در ریشه‌ی اصلی برنامه‌ی شما باشند.

## ۴: مشاهده خروجی

بعد از انجام مراحل قبلی و اولین استقرار، فینگ برای برنامه‌ی شما زیر دامنه‌ای اختصاص می‌دهد که می‌توانید از طریق آن از صحت استقرار برنامه در سرویس**فینگ** مطمئن شوید.

زیر دامنه‌ی شما به این صورت خواهد بود: `https://node-app.apps.fing.ir`
